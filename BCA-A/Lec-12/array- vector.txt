What is an Array?

An array is a collection of variables of the same type stored in contiguous memory.

It has a fixed size, decided at the time of declaration.

üîπ Array Syntax
type arrayName[size];

üîπ Example
#include <iostream>
using namespace std;

int main() {
    int arr[5];  // array of 5 integers

    // taking input
    for(int i = 0; i < 5; i++) {
        cin >> arr[i];
    }

    // printing array
    for(int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    return 0;
}

Key Points about Arrays:

Index starts from 0 ‚Üí arr[0] is first element.

Size cannot change after declaration.

Use loops to traverse or modify array.

Example Problem: Sum of Array
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int sum = 0;

    for(int i = 0; i < 5; i++) {
        sum += arr[i];
    }

    cout << "Sum = " << sum << endl;
    return 0;
}

What is a Vector?

A vector is a dynamic array in C++.

Size can increase or decrease at runtime.

Part of the STL (Standard Template Library).

üîπ Vector Syntax
#include <vector>
vector<int> v;   // vector of integers

üîπ Example: Basic Operations
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> v;

    // Adding elements
    v.push_back(10);
    v.push_back(20);
    v.push_back(30);

    // Access elements
    for(int i = 0; i < v.size(); i++) {
        cout << v[i] << " ";
    }
    cout << endl;

    // Remove last element
    v.pop_back();

    // Print again
    for(int x : v) {
        cout << x << " ";
    }
    cout << endl;

    return 0;
}

üîπ Key Points about Vectors:

Can grow or shrink dynamically ‚Üí push_back(), pop_back().

Access elements like array ‚Üí v[i].

Can use range-based for loop ‚Üí for(int x : v).

Part of STL, so include #include <vector>.
